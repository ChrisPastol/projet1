UX Design
UX Design signifie User Experience Design. En Français,
il s’agit du design de l’expérience utilisateur.
A ne pas confondre avec l’UI Design

UI Design
UI est l’abréviation d’user interface ou interface utilisateur.
L’UI design se rapporte donc à l’environnement graphique dans lequel évolue l’utilisateur d’un logiciel,
d’un site web ou d’une application

SEO
L’optimisation pour les moteurs de recherche, référencement naturel ou SEO (pour search engine optimization),
est un ensemble de techniques visant à optimiser la visibilité d'une page web dans les résultats de recherche
